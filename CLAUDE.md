# 프로젝트 규칙

## 마크다운 링크 출력 규칙

스킬/가이드 파일에서 URL을 출력할 때 반드시 지켜야 하는 규칙:

1. 모든 URL은 코드 블록 밖에서 마크다운 링크 형식 `[텍스트](URL)` 으로 표시한다
2. 코드 블록 안에 URL을 넣지 않는다 (클릭 불가, 줄바꿈 발생). 단, JSON 설정 예시의 URL 값은 설정값이므로 코드 블록 안에 표기한다
3. **링크 닫는 괄호 `)` 뒤에 반드시 공백을 한 칸 넣는다** — 한글 조사/글자가 링크에 포함되어 404가 발생하는 것을 방지

```
[페이지](URL) 에서 확인하세요   ← O (공백 있음)
[페이지](URL)에서 확인하세요    ← X (404 발생)
```
